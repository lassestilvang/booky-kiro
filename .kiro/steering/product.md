# Product Overview

Bookmark Manager Platform is a production-ready, cross-platform bookmark management system with comprehensive organization, archiving, search, and collaboration capabilities.

## Core Features

- Save and organize web bookmarks with rich metadata
- Hierarchical collections with custom icons
- Multi-tag support with advanced filtering
- Browser extension for quick saving (Chrome, Firefox, Safari, Edge)
- Import/export from other bookmark managers
- Cross-device real-time synchronization

## Pro Features

- Full-text search within saved page content
- Permanent archival with page snapshots for offline access
- Highlights and annotations on saved pages
- Collaboration with sharing capabilities (specific users or public)

## Target Users

Users who need robust bookmark management beyond basic browser capabilities, with emphasis on archival, search, and collaboration.
